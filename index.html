م2-3 — الملفات والمجلدات

🗂️ أوامر التنقل والعرض
الأمر الوظيفة مثال
pwd عرض المسار الحالي pwd
ls عرض محتويات المجلد ls -la
ls -l عرض تفصيلي ls -l /etc
ls -a عرض الملفات المخفية ls -a
ls -R عرض متعدد المستويات ls -R /home
cd تغيير المجلد cd /home/ali
cd .. الرجوع لمجلد أعلى cd ..
cd ~ الذهاب للمجلد الرئيسي cd ~
⚠️ المسار المطلق vs النسبي
المسار المطلق — يبدأ بـ /
cd /home/ali/Y
يعمل من أي مكان
المسار النسبي — لا يبدأ بـ /
cd Y
يعمل من المجلد الحالي فقط
ملاحظة: الـ / (slash) تفصل أسماء المجلدات  |  الـ \ (backslash) للاستمرار في السطر التالي
📂 إنشاء وحذف المجلدات
الأمر الوظيفة
mkdir X إنشاء مجلد X
mkdir -p X/one إنشاء مجلدات متداخلة (يُنشئ X وone معاً)
mkdir -v X إنشاء مع إظهار رسالة
rmdir X حذف مجلد فارغ فقط
rmdir -p X/one حذف مجلدات متداخلة فارغة
rm -r X حذف مجلد وكل محتوياته
rm -r -i /home/ali حذف مع سؤال قبل كل ملف
⚠️ خطأ شائع: rmdir لا يحذف إلا المجلد الفارغ — استخدم rm -r للمجلدات التي تحتوي ملفات
📄 إنشاء الملفات والكتابة فيها (cat)
cat > X/one/test Hello World Ctrl+D (لحفظ وإغلاق)

← إنشاء ملف جديد أو استبدال موجود

cat >> ~/test إضافة نص Ctrl+D

← إضافة إلى ملف موجود (لا تمسح محتواه)

cat X/one/test # عرض الملف cat file1 > file2 # نسخ file1 إلى file2 cat file1 >> file2 # إضافة file1 إلى file2
🔀 التحويل (Redirection) والـ Pipeline
الرمز الاستخدام مثال
> تحويل المخرجات إلى ملف (يمسح القديم) ls /etc > com
>> إضافة المخرجات لملف (لا يمسح) ls >> com
< إدخال من ملف بدل لوحة المفاتيح mail ali < com
| Pipeline — مخرج أمر يصبح دخل التالي ls -l | tee ls
1> تحويل المخرج القياسي فقط ls 1>com
2> تحويل الأخطاء فقط ls 2>error
ls -l /etc 2>error 1>com # أخطاء في error، نتائج في com ls -l | tee ls # يعرض ويحفظ في نفس الوقت tee -a # يضيف بدل أن يستبدل
📋 نسخ ونقل الملفات
الأمر الوظيفة
cp test1 test2 نسخ ملف
cp -R X Y/XX نسخ مجلد كامل
mv test1 test2 نقل أو إعادة تسمية
mv file1 file2 dir/ نقل عدة ملفات لمجلد
cp ali/X/one/test ali/Y/test2 # نسخ ملف باسم جديد cp -R ali/X ali/Z/XX # نسخ مجلد X باسم XX داخل Z mv test2 exam # إعادة تسمية test2 إلى exam
ملاحظة mv: إذا كان arg2 مجلداً موجوداً → ينقل إليه | إذا لم يكن موجوداً → يعيد التسمية

م4 — الصلاحيات chmod وumask

📊 جدول الصلاحيات (احفظ هذا!)
r
قراءة (Read)
w
كتابة (Write)
x
تنفيذ (Execute)
-
بدون صلاحية
الصلاحية User (u) Group (g) Other (o)
r (قراءة) 400 40 4
w (كتابة) 200 20 2
x (تنفيذ) 100 10 1
1️⃣ الطريقة الرمزية
chmod u=rwx,g=rx,o=rx X/one/test chmod go-w X/one/test chmod a+x X/one/test chmod +x X/one/test
رمز معنى
u المالك (User)
g المجموعة (Group)
o الآخرون (Other)
a الكل (All)
+ إضافة صلاحية
- إلغاء صلاحية
= تحديد صلاحيات بالضبط
2️⃣ الطريقة العددية — أمثلة
chmod 755 = 400+200+100+40+10+4+1 = rwxr-xr-x
chmod 644 = 400+200+40+4 = rw-r--r--
chmod 766 = 400+200+100+40+20+4+2 = rwxrw-rw-
chmod 744 = 400+200+100+40+4 = rwxr--r--
chmod 755 X/one/test chmod 644 Y/exam
3️⃣ النظام الثُّلاثي (Fibonacci / Octal)

كل صلاحية تُمثَّل بـ 3 بتات ثنائية: rwx = 111 = 7

مثال: rwx r-- r-- ثنائي: 111 100 100 عشري: 7 4 4 → chmod 744
rwx ثنائي عشري
rwx 111 7
rw- 110 6
r-x 101 5
r-- 100 4
-wx 011 3
-w- 010 2
--x 001 1
--- 000 0
🎭 umask — قناع القيم الافتراضية
الصلاحيات الافتراضية للملفات
666
الصلاحيات الافتراضية للمجلدات
777
القاعدة: الصلاحية الفعلية = القيمة الافتراضية − umask
مثال: umask 026 → الملف = 666 - 026 = 640
umask # عرض قيمة umask الحالية umask 026 # تغيير umask umask 0022 # النظام الافتراضي عادةً
ملاحظة: umask يؤثر على الملفات والمجلدات معاً بنفس القيمة
🔑 passwd — تغيير كلمة المرور
passwd # Changing password for username: # Enter old password: # New password: # Re-enter new password:
  • يُخزَّن في /etc/passwd (مشفر في /etc/shadow)
  • كلمة المرور لا تظهر عند الكتابة
  • يجب أن تكون طويلة وتحتوي أرقاماً وحروفاً
👁️ more — عرض الملفات صفحة صفحة
more filename more -22 filename # 22 سطر لكل صفحة
المفتاح الوظيفة
Space الصفحة التالية
Enter سطر واحد للأمام
b الصفحة السابقة
d أو Ctrl+D نصف صفحة للأمام
q أو Q الخروج
/text البحث عن نص
-f إخفاء + عرض رقم السطر

م5 — الأوامر المتقدمة

📊 wc — إحصاء الكلمات والأسطر
wc filename wc -l filename # عدد الأسطر wc -w filename # عدد الكلمات wc -c filename # عدد الأحرف (bytes)

يعطي 3 أرقام: عدد الأسطر، الكلمات، الأحرف

✂️ cut — قطع أجزاء من الأسطر
cut -f 1,3 X/one/test # قطع العمود 1 و3 (فصل بـ tab) cut -c 1-8 X/one/test # قطع الأحرف من 1 إلى 8 cut -f 1-3,10 file # قطع أعمدة 1 إلى 3 والعمود 10
الخيار الوظيفة
-f قطع بالعمود (مفصولة بـ tab افتراضياً)
-c قطع بالأحرف
-d تحديد محدد مختلف (delimiter)
🔎 grep — البحث في الملفات
grep "john" X/one/test grep "john smith" X/one/test # بحث عن جملة (بين "") grep -i "john" file # بحث غير حساس للحجم grep "pattern" file1 file2 # بحث في عدة ملفات
التعابير النمطية (Regular Expressions):
الرمز المعنى مثال
. أي حرف واحد grep "a.b" file
* صفر أو أكثر من السابق grep "ab*" file
[abc] أي حرف من المجموعة grep "[aeiou]" file
[a-z] نطاق من a إلى z grep "[a-z]" file
[^abc] أي حرف ليس في المجموعة grep "[^0-9]" file
^ بداية السطر grep "^root" file
$ نهاية السطر grep "bash$" file
\ إلغاء المعنى الخاص للرمز grep "\." file
grep "^[a-f]" file # يبحث عن سطر يبدأ بـ a إلى f grep "[a-f][a-m]" file # حرف من a-f ثم حرف من a-m grep "smith" file1 file2 # بحث في ملفين
👤 who — عرض المتصلين
who who -H # مع عناوين الأعمدة who -g # مع معلومات المجموعة who am i # معلوماتي أنا

يعرض: اسم المستخدم، TTY، وقت الاتصال

📊 w — إحصاءات المستخدمين
w w username

يعرض: USER, TTY, FROM, LOGIN@, IDLE, JCPU, PCPU, WHAT

  • IDLE — وقت الخمول
  • JCPU — وقت المعالج للجلسة
  • PCPU — وقت المعالج للعملية الحالية
  • WHAT — الأمر الحالي
💤 sleep — تأخير الأوامر
sleep 60 # ينتظر 60 ثانية sleep 60 ; ls # ينفذ ls بعد دقيقة
🔗 alias — اختصارات الأوامر
alias ls="ls -l" alias rm="rm -i" alias # عرض كل الـ aliases
💬 التواصل بين المستخدمين
الأمر الاستخدام الخروج
write ali إرسال رسالة فورية لمستخدم Ctrl+D
mesg y/n السماح/منع استقبال الرسائل —
mail ali إرسال بريد (يُخزَّن في /var/spool/mail) Ctrl+D أو .
mail قراءة البريد الوارد q
write ali # يكتب رسالة لـ ali mail ali < com # يرسل محتوى الملف com بريداً لـ ali mail -s "subject" ali # بريد بعنوان

م8 — إدارة المستخدمين والمجموعات

➕ useradd — إضافة مستخدم جديد
/usr/sbin/useradd options username
الخيار الوظيفة مثال
-m إنشاء مجلد المستخدم تلقائياً useradd -m ali
-d path تحديد مجلد رئيسي مختلف useradd -d /home/mydir ali
-s shell تحديد نوع shell useradd -s /bin/bash ali
-G group إضافة لمجموعة إضافية useradd -G floppy ali
-e date تاريخ انتهاء الحساب useradd -e 2020-05-15 ali
-p pass كلمة مرور مشفرة useradd -p "crypt" ali
-f days أيام قبل تعطيل الحساب useradd -f 30 ali
-c comment تعليق (الاسم الكامل) useradd -c "Ali Ahmed" ali
useradd -s /bin/tcsh -G floppy -e 2020-05-15 ali # ثم تعيين كلمة مرور passwd ali
➖ userdel — حذف مستخدم
/usr/sbin/userdel username # حذف المستخدم فقط /usr/sbin/userdel -r username # حذف المستخدم + مجلده + ملفاته
⚠️ الخيار -r يحذف كل ملفات المستخدم نهائياً!
➕ groupadd
/usr/sbin/groupadd options groupname groupadd -g 500 developers # مع تحديد GID
➖ groupdel
/usr/sbin/groupdel groupname

لا يحذف المستخدمين داخل المجموعة

⚙️ groupmod — تعديل المجموعة
/usr/sbin/groupmod options groupname
الخيار الوظيفة
-n newname تغيير اسم المجموعة
-g newGID تغيير رقم GID
🔑 gpasswd — إدارة كلمة مرور المجموعة
gpasswd options group-name
الخيار الوظيفة
-a user إضافة مستخدم للمجموعة
-d user حذف مستخدم من المجموعة
-r حذف كلمة المرور
-R تعطيل الوصول للمجموعة
🔄 newgrp — تغيير المجموعة الحالية
newgrp new-group-name

يغير المجموعة الفعّالة للمستخدم في الجلسة الحالية

🧠

اختبر نفسك

التقدم 0 / 10
1. ما هي نتيجة الأمر: chmod 644 file ؟
644 = 400+200+40+4 = rw-r--r-- | User: rw (لا تنفيذ) | Group: r-- | Other: r--
2. كم حقلاً في ملف /etc/shadow؟
shadow = 9 حقول | passwd = 7 حقول | group = 4 حقول
3. ما الفرق بين > و >> ؟
> يستبدل الملف كاملاً | >> يضيف في نهاية الملف الموجود
4. ما قيمة umask 026 على الملفات؟ (الافتراضي 666)
666 - 026 = 640 (rw-r-----) | User: rw | Group: r | Other: لا شيء
5. الحقل السادس في /etc/passwd هو؟
passwd: username:pass:UID:GID:comment:HOME:shell | الحقل 6 = home | الحقل 7 = shell
6. ما الخيار الذي يُنشئ المجلد الرئيسي تلقائياً عند useradd؟
-m = make home | -d = تحديد مسار المجلد | -G = المجموعة الإضافية | -s = نوع shell
7. ما الأمر الصحيح لإنشاء مجلدات X/one/two دفعة واحدة؟
-p تُنشئ المجلدات الأب إذا لم تكن موجودة | بدون -p سيفشل إذا لم يكن X موجوداً
8. grep "^[a-f]" file — ماذا يبحث؟
^ = بداية السطر | [a-f] = أي حرف من a إلى f | الاثنان معاً = سطر يبدأ بحرف من a إلى f
9. ما الأمر الصحيح لحذف مجلد مع كل محتوياته؟
rm -r = حذف متعدد المستويات | rmdir = للمجلدات الفارغة فقط | rm بدون -r = للملفات فقط
10. chmod 755 — ما الصلاحيات الممنوحة للمجموعة؟
755: User=7(rwx) | Group=5(r-x) | Other=5(r-x) | 5 = 4+1 = r-x (قراءة + تنفيذ)
